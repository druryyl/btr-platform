## 1. Implementation Summary

Delivered **Supplier** as the third Entity Analytics consumer by mirroring the M32.9 Salesman entity-pack pattern: registrar, attention/relationship catalogs, producer (L0–L5 orchestration), worker hook on `PurchasingManagement`, PU01 `ProfileRoute` links, frontend profile/compare routes, and reconciliation tests — with **no platform engine or L0–L5 schema changes**.

```text
RefreshDashboardPurchasingManagementSnapshotWorker
  → DashboardPurchasingManagementAggregator (+ Portfolio rows)
  → ISupplierMtdItemRollupDal + ISupplierDal
  → DashboardSupplierRelationshipAggregator
  → BTRPD_PurchasingManagement snapshot save
  → EntityAnalyticsProducerOrchestrator
        → SupplierEntityAnalyticsProducer
              L0 CURRENT  (EntityId = SupplierId, EntityCode = SupplierCode)
              L1 MONTHLY  (TrendEligible KPIs)
              L2 RANKING  (RankEligible KPIs; peer = supplier-all-active)
              L3 ATTENTION (M21 signal keys via EntityAttentionEngine)
              L4 RELATIONSHIP (Top customers/salesmen/products)
              L5 RADAR     (peer group supplier-all-active)
  → GET profile / search / compare (existing platform API + Vue shells)
```

## 3. Producer Design (L0–L5)

| Layer | Source | Entity keys |
| ----- | ------ | ----------- |
| **L0** | `Portfolio` rows (active principals with MTD purchase, inventory, or sales-out) | `EntityId = SupplierId`, `EntityCode = SupplierCode` |
| **L1** | TrendEligible KPI values for current `(Year, Month)` | Must match PU01 principal exposure row for same supplier/period |
| **L2** | RankEligible KPIs across active suppliers | Peer group `supplier-all-active` |
| **L3** | `AttentionList` grouped by `SupplierId` (principal rows only) | M21 signal constants |
| **L4** | `DashboardSupplierRelationshipAggregator` output | Pack `supplier-relationships` |
| **L5** | `PeerGroupRuleId = supplier-all-active` | Six radar axes from registrar metadata |

**KPI mapping (L0 numeric rows)**

| KPI ID | Portfolio field |
| ------ | ----------------- |
| PU-KPI-001 | `MtdPurchaseAmount` |
| PU-KPI-002 | `MtdInvoiceCount` |
| PU-KPI-003 | `PostedPercent` |
| EA-DIM-INVENTORY-VALUE | `InventoryValue` |
